# The Meals App

This project was bootstrapped with [Expo](https://expo.io/).

## How to use it?

### Download manually

Install:

```bash
yarn install
# or
npm install
```

Run:

```bash
yarn start
# or
npm run start
```

Opening an iOS simulator:

```bash
yarn ios
# or
npm run ios
```

Opening an Android simulator:

```bash
yarn android
# or
npm run android
```

Opening a browser:

```bash
yarn web
# or
npm run web
```

Prettier:
```bash
yarn prettier
# or
npm run prettier
```

Eject:
```bash
yarn eject
# or
npm run eject
```
